API Documentation

Propel Career API v1.0.0 · OpenAPI 3.0 · Public endpoint

10 endpoints Swagger UI
🔐 Most endpoints require a Clerk Bearer token. Paste your session token using the Authorize button.
📄 Resume
POST /api/parse-resume Resume 🔐 Auth Upload and parse a PDF resume
Request: multipart/form-data · Field: resume (binary PDF)
Response 200: { text: string, fileName: string }
Response 400: Invalid file (not a PDF)
Response 401: Unauthorized
POST /api/generate-resume Resume 🔐 Auth Generate updated resume PDF
Request: { resumeText: string, acceptedSuggestions: Suggestion[] }
Response 200: application/pdf binary stream
Response 401: Unauthorized
🔍 Analysis
POST /api/analyze-match Analysis 🔐 Auth Get ATS match score
Request: { resumeText: string (min 50), jobDescription: string (min 50) }
Response 200: { score: number, matchedKeywords: string[], missingKeywords: string[] }
Response 401: Unauthorized  Response 402: Insufficient credits
POST /api/suggest-improvements Analysis 🔐 Auth Get AI wording suggestions
Request: { resumeText: string, jobDescription: string }
Response 200: { suggestions: Suggestion[] }
Response 401: Unauthorized  Response 402: Insufficient credits
💳 Payments
POST /api/stripe/checkout Payments 🔐 Auth Create Stripe Checkout session
Request: { packId: "starter" | "pro" }
Response 200: { url: string } — Stripe hosted checkout URL
Response 401: Unauthorized
POST /api/stripe/webhook Payments Stripe webhook (Stripe-signed · public)
Request: Raw Stripe event body · Header: stripe-signature
Response 200: { received: true }
Response 400: Invalid signature
👤 User
GET /api/user/credits User 🔐 Auth Get current credit balance
Response 200: { credits: number }
GET /api/user/account User 🔐 Auth Full account data incl. usage history
Response 200: { credits, status, thresholds: {reminder, warning}, creditsUsed, recentUsage[] }
GET /api/user/usage User 🔐 Auth Usage event history with LLM detail
Response 200: { events: UsageEvent[] } — last 50, includes nested llmUsage
🛡 Admin
GET /api/admin/usage Admin 🔐 Admin only Aggregated feature + LLM cost stats
Response 200: { featureStats: {feature, count, avgMs}[], llmStats: {provider, model, operation, tokens, costUsdMicros}[] }
Response 403: Forbidden (non-admin)
OpenAPI spec available as JSON at GET /api/docs · Powered by @asteasolutions/zod-to-openapi
